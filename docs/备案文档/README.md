## 使用说明

- 提交 ICP 备案、公安备案或应用上架审核时，可从上述文档摘录「功能简介」「服务说明」等字段。
- 文档中的 `【备案域名】`、主体名称、联系方式等占位项，请按实际备案信息替换后再使用。
- 新增备案相关材料（如隐私政策、用户协议、安全评估说明等）请统一放在本目录，并更新本索引。
